# GPT2 Text Generation

This project demonstrates text generation using a fine-tuned GPT-2 model. The model is trained on a custom dataset to generate coherent and context aware text based on a given prompt.

## Features

* Fine-tuning GPT-2 on custom text data
* Generates human like text
* Simple and modular implementation
* Built using Hugging Face Transformers and PyTorch

## Tech Stack

* Python
* Transformers (Hugging Face)
* PyTorch
* Datasets

## Project Structure

GPT2 Text Generation/
- data.txt
- train.py
- generate.py
- requirements.txt
- model/

## Installation

pip install -r requirements.txt

## Train the model

python train.py

## Generate text

python generate.py
